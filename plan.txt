# Kanban Dashboard — AI News Feed Feature Plan
# Generated: 2026-03-30

## Goal
Daily agentic/AI news feed visible in the dashboard. Fetcher runs every morning via cron,
stores articles in DB, exposed via API, displayed in a dedicated NewsPage.

## Architecture
- fetch-news.js  : standalone Node.js script — fetches 5 RSS feeds, parses, upserts to ai_news table
- server.js      : GET /api/news + POST /api/news/refresh (spawns fetch-news.js)
- NewsPage.jsx   : dedicated page — two tabs (Agentic AI / All AI), article cards with title link + source + date + summary
- App.jsx        : add showNews state to switch between ProjectsPage and NewsPage
- ProjectsPage   : add "AI News" button in header
- Cron job       : 0 7 * * * on server runs fetch-news.js every morning

## RSS Sources (no API key needed)
1. TechCrunch AI   – https://techcrunch.com/tag/artificial-intelligence/feed/
2. VentureBeat AI  – https://venturebeat.com/category/ai/feed/
3. The Verge AI    – https://www.theverge.com/rss/ai-artificial-intelligence/index.xml
4. MIT Tech Review – https://www.technologyreview.com/feed/
5. HuggingFace     – https://huggingface.co/blog/feed.xml

2. rsync + restart kanban-api (no new npm deps — only mysql2 + built-in https)
3. Run fetch-news.js once manually to seed data
4. Add cron: 0 7 * * * /usr/bin/node /home/rajramani/kanban-api/fetch-news.js >> /home/rajramani/kanban-api/news-fetch.log 2>&1
